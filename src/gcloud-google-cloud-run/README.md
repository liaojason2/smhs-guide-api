# HTTP server for deploy to Google Cloud Run

## Quickstart

- Run server on localhost

```sh
# Install dependency
npm install

# Run Server
npm start
```

- Deploy to GCR

```sh
# Deploy to Google Cloud Run
gcloud run deploy
```
